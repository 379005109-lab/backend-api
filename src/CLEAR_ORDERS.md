# 清空假订单数据

## 说明

此脚本用于清空数据库中的所有假订单数据，为真实订单数据做准备。

## 使用方法

在 `server` 目录下运行：

```bash
npm run clear-orders
```

或者：

```bash
node src/scripts/clearFakeOrders.js
```

## 功能

- 查找所有订单数据
- 显示订单列表（订单号、用户、总额、状态）
- 删除所有订单数据
- 清空后可以创建真实订单

## 注意事项

⚠️ **警告**：此操作会删除数据库中的所有订单数据，请谨慎操作！

- 执行前请确保已备份重要数据
- 此操作不可逆
- 建议在测试环境先执行验证

## 执行结果

脚本执行后会显示：
1. 找到的订单数量
2. 订单列表详情
3. 删除的订单数量
4. 确认信息
